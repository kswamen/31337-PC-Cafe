# 31337-PC-Cafe

### PC 카페 상품 주문 프로그램입니다.

[작동 영상 링크는 여기를 클릭하세요.](https://blog.naver.com/kswamen/222186239584)

## 사용 및 설치법:


### 본 프로그램에서 사용되는 음성 파일과 이미지 파일을 사용하기 위해, C:\ 위치에 다음과 같이 폴더를 생성해 주세요.

```sh
├─C:\
  │  
  ├─31337
  │      image
  └─     voice

```

### 본 프로그램에서는 데이터베이스로 MariaDB를 사용하고 있습니다. 아래 이미지를 참고해 데이터베이스를 생성해 주세요.
